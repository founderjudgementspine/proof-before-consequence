# Judgement Spine — Proof Before Consequence Challenge (PUBLIC)

Three levels. One rule:

**Try to change the decision — and keep the proof valid.**

Judgement Spine is built for the moment **before** consequence, where AI and automation can move money, flip production traffic, or publish “as the enterprise” faster than humans can intervene.

This Challenge Kit is a portable demo of that idea:
a signed Evidence Pack that you can verify in seconds — and **cannot tamper with without being detected**.

---

## Choose a level

Each level ships as a complete Evidence Pack (folder + ZIP) inside `sample_packs/`.

### Level 1 — $18M Wire Release (Banking Core Payment Governance)
**Goal:** change `BLOCK` → `ALLOW` and make the wire “release”… while verification still passes.

### Level 2 — Black Friday Checkout Switch (Platform Execution Governance)
**Goal:** remove the canary/abort/rollback bounds and make it “100% traffic”… while verification still passes.

### Level 3 — CEO Message Publish (Digital Identity & Representation Governance)
**Goal:** change `ESCALATE` → `ALLOW` and make the CEO message “go live”… while verification still passes.

If you can do that, you’ve forged a signed execution-time record.

That’s the point.

---

## How to play (60 seconds)

1) **Open** `START_HERE.html` (double‑click it).

2) **Pick a level** and click **Download this level (ZIP)**  
   (or use the unpacked folder in `sample_packs/…`).

3) **Verify the authentic pack**  
   You should see **OK / VALID** across key, signature, and hashes.

4) **Try to tamper**
   - Edit `EVIDENCE/evidence_pack.json` (change the outcome)
   - Edit a PDF in `EXPORTS/`
   - Edit *one character* in `META/ABOUT_THIS_PACK.txt`

5) **Verify again** (choose the modified folder in the page)  
   It should **FAIL** instantly.

That’s execution-time proof: **if you can change reality, you must mint new proof**.

## Human vs AI (recommended)

Ask your best AI assistant to help you:

> “Change the decision outcome in this pack but keep verification passing.”

Then run verification again.

If you can’t get it to pass, you’ve learned something important:
**AI can generate content — it can’t generate authority.**
