# Publish Guide (GitHub • Website • LinkedIn)

This kit is already in “drop‑in” format: one folder, one `START_HERE.html`, three levels.

You can publish it in minutes.

---

## 1) Publish on GitHub (repo + GitHub Pages)

### A) Create the repo
1. Create a new GitHub repository (public).
2. Upload everything in this folder.

### B) Turn on GitHub Pages
1. Go to **Settings → Pages**
2. Source: **Deploy from a branch**
3. Branch: **main** / root
4. Save

### C) Share the link
Your one‑click entrypoint is:

`START_HERE.html`

(Hosted by GitHub Pages, it becomes a clean, zero‑install interactive demo.)

---

## 2) Publish on your website (download + one‑click)

Recommended:
- Offer the ZIP as a single download
- Link directly to GitHub Pages for the live interactive page (optional)

### A) “Download” link
Upload the whole kit as a ZIP and link it as:

**Proof Before Consequence Challenge (ZIP)**

Host `START_HERE.html` as a static page and link it as:
